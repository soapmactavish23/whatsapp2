# WhatsApp Clone

## Descrição
WhatsApp Clone é um App para Android, ele é um replica do app WhatsApp com quase todas as funcionalidades do app real, sendo elas: enviar Mensagem, criar grupos, enviar fotos e modificar o perfil.
